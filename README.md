## 🚀 Architecture Overview

This site is statically generated utilizing the **Astro** web framework. By shipping zero unnecessary JavaScript to the client, we achieve bare-metal rendering speeds while preserving the complex, dynamic video overlays and UI/UX designed for our enterprise, defense, and biomedical partners.

* **Framework:** [Astro](https://astro.build/) (v6.4+)
* **Styling:** Custom CSS / Flexbox (Responsive & Mobile-Optimized)
* **CI/CD Pipeline:** Automated via GitHub Actions
* **Hosting:** GitHub Pages Edge Network
